Tentang Penelitian

BananaDetect – Sistem Klasifikasi Tingkat Kematangan Buah Pisang Menggunakan ResNet50 dan VGG16.

📌 Latar Belakang

Buah pisang (Musa paradisiaca) merupakan salah satu komoditas hortikultura terpenting di Indonesia. Penentuan tingkat kematangan pisang secara manual memerlukan keahlian khusus dan rentan terhadap kesalahan subjektif. Proses ini juga membutuhkan waktu yang tidak efisien dalam skala industri.

Kemajuan teknologi Deep Learning, khususnya Convolutional Neural Network (CNN), membuka peluang besar untuk otomasi klasifikasi kematangan buah berdasarkan citra digital. Pendekatan ini dapat meningkatkan efisiensi, konsistensi, dan akurasi penilaian kematangan pisang.

🎯 Tujuan Penelitian

  • Membangun sistem klasifikasi tingkat kematangan pisang berbasis Deep Learning yang akurat dan dapat diimplementasikan secara praktis.
  • Membandingkan performa dua arsitektur CNN populer (VGG16 dan ResNet50) menggunakan metode transfer learning pada dataset pisang.
  • Mengembangkan pipeline preprocessing gambar yang optimal termasuk ekstraksi fitur warna RGB, HSV, dan tekstur GLCM.
  • Menyediakan aplikasi web berbasis Flask untuk prediksi kematangan pisang secara real-time.

🧪 Metode Penelitian

Penelitian ini menggunakan pendekatan Transfer Learning dengan dua tahap utama:

  • Feature Extraction: Menggunakan bobot pre-trained ImageNet pada layer konvolusi untuk mengekstrak fitur visual umum dari gambar pisang.
  • Fine-tuning: Melatih ulang classifier head (GlobalAveragePooling → Dense 256 → Dropout → Softmax 3) dengan dataset pisang spesifik.

🧠 Convolutional Neural Network (CNN)

CNN adalah arsitektur jaringan saraf tiruan yang dirancang khusus untuk memproses data grid seperti gambar. CNN terdiri dari tiga jenis layer utama:

Convolutional Layer
Mengekstrak fitur lokal seperti tepi, tekstur, dan pola menggunakan filter/kernel.
Pooling Layer
Mengurangi dimensi feature map sambil mempertahankan informasi penting (downsampling).
Fully Connected Layer
Menggabungkan seluruh fitur untuk klasifikasi final menggunakan fungsi aktivasi Softmax.

🔄 Transfer Learning

Transfer learning adalah teknik memanfaatkan pengetahuan yang telah dipelajari dari tugas/dataset besar (ImageNet – 1.2 juta gambar, 1000 kelas) untuk menyelesaikan tugas yang berbeda namun berkaitan. Pendekatan ini sangat efektif ketika dataset target relatif kecil.

ImageNet (1.2M img) → Pre-trained Weights → VGG16/ResNet50
→ Freeze Base Layers → Add Custom Head → Fine-tune → Banana Dataset

🗂️ Dataset

Dataset yang digunakan adalah Banana Classification Dataset dari Kaggle (atrithakar), terdiri dari 963 gambar pisang yang terbagi dalam 3 kategori kematangan.
